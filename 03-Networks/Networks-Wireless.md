## 1. Wireless Networking Kya Hai

- **Bina cable ke** network connection
- Radio waves se data transfer hota hai
- Aajkal sabse zyada use hota hai!

```
Router → Radio Waves → Device
  ↑                      ↓
  └──── Data ────────────┘
```

## 2. WiFi Standards (802.11)

| Standard | Name | Speed | Frequency | Year |
|----------|------|-------|-----------|------|
| **802.11b** | WiFi 1 | 11 Mbps | 2.4 GHz | 1999 |
| **802.11g** | WiFi 3 | 54 Mbps | 2.4 GHz | 2003 |
| **802.11n** | WiFi 4 | 600 Mbps | 2.4/5 GHz | 2009 |
| **802.11ac** | WiFi 5 | 3.5 Gbps | 5 GHz | 2013 |
| **802.11ax** | WiFi 6 | 9.6 Gbps | 2.4/5/6 GHz | 2019 ✅ |
| **802.11ax** | WiFi 6E | 9.6 Gbps | 6 GHz | 2021 |

## 3. 2.4 GHz vs 5 GHz

| | 2.4 GHz | 5 GHz |
|--|---------|-------|
| **Range** | Zyada ✅ | Kam |
| **Speed** | Slow | Fast ✅ |
| **Interference** | Zyada | Kam |
| **Wall penetration** | Better ✅ | Weak |
| **Devices** | Zyada support | Kam |
| **Use** | Large area, walls | Fast speed, close range |

```
2.4 GHz → Deewar ke paar bhi signal ✅ (but slow)
5 GHz   → Bahut fast ✅ (but kam range)
```

## 4. WiFi Security Protocols

| Protocol | Security Level | Use |
|----------|---------------|-----|
| **WEP** | Bahut weak ❌ | Avoid karo |
| **WPA** | Medium | Old, avoid |
| **WPA2** | Strong ✅ | Common |
| **WPA3** | Bahut Strong ✅ | Latest |

```
WEP  → Easily hackable, avoid ❌
WPA  → Better but old
WPA2 → AES encryption, common ✅
WPA3 → Sabse secure aajkal ✅
```

### Wireless Security Tips:
```
✅ WPA2 ya WPA3 use karo
✅ Strong password rakho (12+ characters)
✅ Default router password change karo
✅ Guest network alag rakho
✅ Regular firmware update karo
❌ Public WiFi pe sensitive kaam mat karo
❌ WEP bilkul use mat karo
❌ Open network mat rakho
```

## 5. Wireless Network Types

### 1. Infrastructure Mode:
```
Device → Access Point → Router → Internet
(Sabse common - ghar, office) ✅
```

### 2. Ad-hoc Mode:
```
Device ←→ Device (Direct connection)
(Bina router ke - rare use)
```

### 3. Hotspot:
```
Phone → Mobile Data → WiFi Hotspot → Other Devices
(Jab apna internet doosron ke saath share karo)
```

### 4. Mesh Network:
```
Router ←→ Node 1 ←→ Node 2 ←→ Node 3
(Poore ghar mein WiFi coverage)
Example: Google Nest, TP-Link Deco
```

## 6. Bluetooth

| Feature | Detail |
|---------|--------|
| **Range** | ~10 meter |
| **Speed (Classic)** | 3 Mbps |
| **Speed (BLE)** | 2 Mbps |
| **Use** | Headphones, Mouse, Keyboard, Speaker |
| **Latest Version** | Bluetooth 5.3 |

### Bluetooth vs WiFi:

| | Bluetooth | WiFi |
|--|-----------|------|
| Range | 10m | 35m+ |
| Speed | 3 Mbps | Gbps |
| Power | Kam ✅ | Zyada |
| Use | Personal devices | Internet |

### BLE (Bluetooth Low Energy):
```
✅ Bahut kam battery use karta hai
✅ IoT devices mein use hota hai
✅ Smartwatch, fitness tracker
```

## 8. Quick Revision

```
WiFi          = Wireless network (Radio waves)
802.11ax      = WiFi 6, latest standard ✅
2.4 GHz       = Long range, slow speed
5 GHz         = Short range, fast speed
WPA2/WPA3     = WiFi security protocols ✅
WEP           = Old, weak, avoid ❌

Infrastructure = Normal WiFi (router + devices)
Ad-hoc        = Direct device to device
Hotspot       = Phone se internet share
Mesh          = Multiple nodes, full coverage

Bluetooth     = Short range (10m), low power
BLE           = Bluetooth Low Energy (IoT)
4G/LTE        = Common mobile internet ✅
5G            = Future, ultra fast
```

### Important Standards:
```
WiFi 4 (802.11n)  → 600 Mbps
WiFi 5 (802.11ac) → 3.5 Gbps
WiFi 6 (802.11ax) → 9.6 Gbps ✅ Latest
```
